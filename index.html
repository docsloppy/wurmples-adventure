Firestore Database Permission Blocked! 🔒

We detected a "Missing or insufficient permissions" error from your Google Firebase project. This happens because your database rules are locked by default. The app has automatically switched to Local Storage Mode so your son can still earn coins and evolve his Wurmple locally in his browser!

Follow these 4 simple steps to fix this and enable Cloud Sync:

  1. Open your Firebase Console.
  2. In the left menu under Build, click Firestore Database, then click the Rules tab at the top.
  3. Copy and paste the exact block below to replace your existing rules:
rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /{document=**} { allow read, write: if request.auth != null; } } }

4. Click the blue "Publish" button at the top-right. Once published, refresh this page and everything will sync automatically! 🎉
